# E-E-A-T Stärkung – Seite Stillen

**Route:** `/phase/breastfeeding`  
**Stand:** Februar 2026

## Übersicht

Die Stillen-Pillar-Seite wurde um E-E-A-T-Elemente (Experience, Expertise, Authoritativeness, Trustworthiness) ergänzt. Änderungen erfolgen ausschließlich inhaltlich – keine Layout-, Design- oder Komponentenänderungen.

---

## Schritt 1 – Experience (Erfahrung subtil integriert)

### Ergänzte Formulierungen

| Abschnitt | Formulierung | Position |
|-----------|--------------|----------|
| Der Stillstart – die ersten Tage | „In der Praxis zeigt sich häufig: In den ersten Tagen kommt der Milcheinschuss …“ | 2. Absatz |
| Häufige Stillprobleme | „Aus Erfahrung in der Begleitung von Familien: Position und Technik zu prüfen lohnt sich …“ | 3. Absatz (wunde Brustwarzen) |
| Häufige Stillprobleme | „Viele Mütter berichten in der Beratung von dieser Unsicherheit – oft zeigt sich …“ | 4. Absatz (zu wenig Milch) |
| Abstillen – ein natürlicher Übergang | „In der Praxis hat sich bewährt: Eine Mahlzeit nach der anderen ersetzen …“ | 2. Absatz |

**Anzahl Experience-Formulierungen:** 4 (in 3 Abschnitten)

### Ton

- Neutral, keine Ich-Form
- Kein Marketing, keine CTA

---

## Schritt 2 – Expertise deutlich gemacht

### Ergänzungen im Abschnitt „Was bedeutet Stillen?“

- **Physiologischer Prozess:** bereits vorhanden (Brustgewebe, Milchgänge, Hormone)
- **Regulation durch Hormone:** erweitert um Hinweis auf Sensibilität gegenüber Stress
- **Bedeutung von Hautkontakt:** ergänzt um „Hautkontakt und frühes Anlegen werden von Fachleitlinien als zentral für einen guten Stillstart eingestuft“
- **Evidenzbasierte Einordnung:** ergänzt „Die hier beschriebenen Zusammenhänge entsprechen dem aktuellen fachlichen Stand“

### Ton

- Neutral, keine Studienaufzählung
- Kein akademischer Overkill

---

## Schritt 3 – Trust-Signal

### Position

**Direkt nach dem FAQ-Block**, vor der NextStepsSection.

### Überschrift

„Fachliche Einordnung“

### Text

> Die hier dargestellten Informationen basieren auf aktuellen fachlichen Erkenntnissen. Jede Stillbeziehung ist individuell. Bei Unsicherheiten oder anhaltenden Beschwerden kann eine persönliche Stillberatung sinnvoll sein.

### Umsetzung

- Hinweis auf evidenzbasierte Inhalte ✓
- Hinweis auf individuelle Situationen ✓
- Empfehlung professioneller Beratung bei Unsicherheiten ✓
- Keine Werbung ✓
- Kein aggressiver Beratungsaufruf ✓

## Schritt 4 – Sprachprüfung

### Durchgeführte Prüfung

| Kriterium | Status |
|-----------|--------|
| Keine Heilsversprechen | ✓ |
| Keine absoluten Aussagen | ✓ |
| Keine Dramatisierung | ✓ |
| Keine medizinischen Übertreibungen | ✓ |
| Keine Schuldzuweisungen | ✓ |

### Anpassungen im Zuge der Prüfung

- „so wertvoll“ → „förderlich“ (weniger absolut)
- „ärztliche Abklärung nötig“ → „ärztliche Abklärung sinnvoll“ (weniger imperativ)
- „Wichtig: rasch handeln“ → „Rasches Handeln kann verhindern …“ (nüchterner)
